# Amazon Cloud Reader

A very simple reader for Amazon Cloud Reader.

## Install:

### Linux

```sh
$ GOOS=linux go build -o cloud-reader cloud_reader.go
```

### Other

```sh
$ go build -o cloud-reader cloud_reader.go
```

## Run

```sh
$ cloud-reader
```
